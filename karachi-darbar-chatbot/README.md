# 🍛 Karachi Darbar Restaurant Chatbot

An AI-powered chatbot for Karachi Darbar Restaurant.
Built with Python and Groq AI (Llama 3.3 70B).

---

## ✨ Features

- Answers questions about Menu, Timing, Location and Booking
- Supports Roman Urdu and English
- Secure API key using .env file
- Runs in terminal

---

## 🛠️ Built With

- Python
- Groq AI API (Llama 3.3 70B)
- python-dotenv

---

## ⚙️ How to Run

### Step 1 — Get a Free Groq API Key
- Go to: https://console.groq.com
- Create a free account
- Copy your API key

### Step 2 — Download the Project
- Click the green "Code" button above
- Select "Download ZIP"
- Extract the ZIP file

### Step 3 — Create .env File
- Create a new file named .env
- Write this inside:
  GROQ_API_KEY=your_api_key_here

### Step 4 — Install Requirements
pip install groq python-dotenv

### Step 5 — Run the Chatbot
python karachi_darbar_chatbot.py

---

## 🔐 API Security

- API key is stored in .env file
- .env is added to .gitignore
- Key is never exposed in code
