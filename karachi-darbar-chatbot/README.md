# 🍛 Karachi Darbar Restaurant Chatbot

An AI-powered bilingual chatbot for Karachi Darbar Restaurant.
Built with Flask and Groq AI (Llama 3.3 70B).

---

## ✨ Features

- Supports Roman Urdu and English
- Provides Menu, Timing, Location and Booking info
- Beautiful dark and light mode UI
- Real-time typing animation
- Quick reply chips for fast answers

---

## 🛠️ Built With

- Python and Flask
- Groq AI API (Llama 3.3 70B)
- HTML, CSS, JavaScript

---

## ⚙️ How to Run

### Step 1 — Get a Free Groq API Key
- Go to: https://console.groq.com
- Create a free account
- Copy your API key

### Step 2 — Download the Project
- Click the green "Code" button above
- Select "Download ZIP"
- Extract the ZIP file on your computer

### Step 3 — Add Your API Key
- Open app.py
- Find this line:
  api_key="YOUR_API_KEY_HERE"
- Replace YOUR_API_KEY_HERE with your Groq API key

### Step 4 — Install Requirements
pip install flask flask-cors groq

### Step 5 — Run the App
python app.py

### Step 6 — Open in Browser
http://localhost:5000
