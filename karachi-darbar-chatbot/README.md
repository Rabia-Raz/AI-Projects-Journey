# 🍛 Karachi Darbar Restaurant AI Chatbot

A full-stack AI-powered chatbot for a restaurant, built with Python (Flask) and Groq API using LLaMA 3.3 70B model.

## ✨ Features

- 🤖 AI-powered responses using LLaMA 3.3 70B via Groq API
- 🌐 Bilingual support — English & Roman Urdu
- ⚡ Real-time responses with typing indicator
- 🎨 Beautiful UI with Dark/Light theme toggle
- 📱 Responsive design — works on mobile too
- 🍽️ Quick chips for Menu, Booking, Timing, Location, Prices

## 🛠️ Tech Stack

- **Backend:** Python, Flask, Flask-CORS
- **AI Model:** LLaMA 3.3 70B (via Groq API)
- **Frontend:** HTML, CSS, JavaScript
- **Other:** Prompt Engineering, REST API

## 🚀 How to Run

**1. Clone the repository:**
```bash
git clone https://github.com/your-username/karachi-darbar-chatbot.git
cd karachi-darbar-chatbot
```

**2. Install dependencies:**
```bash
pip install -r requirements.txt
```

**3. Setup API Key:**
```bash
# .env.example ko copy karke .env banao
cp .env.example .env
# Phir .env mein apni Groq API key daalo
```

**4. Run the app:**
```bash
python app.py
```

**5. Browser mein kholo:**
```
http://localhost:5000
```
